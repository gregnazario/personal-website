Este proyecto es una renovación completa de mi sitio personal. Usualmente hago uno de estos cada 3-5 años para
reaprender algo de trabajo frontend. Aunque, esta vez fue hecho completamente con vibe coding, y algunas
correcciones manuales. Es agradable poder expresarlo simplemente, pero quita algo del aprendizaje
práctico que viene con ello.

Usé Tanstack Start para este proyecto, para evitar Next.js, especialmente después de todas las
vulnerabilidades recientes. Y me aseguré de probar algunos diseños nuevos para hacerlo limpio y agradable, aunque
hay algunos diseños no ideales que pueden mejorarse (y también GPT siempre hace estas páginas raras de
color púrpura neón).
